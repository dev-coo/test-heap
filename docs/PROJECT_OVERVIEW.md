# 프로젝트 개요

## 소개

**MinHeap 시각화**는 Min Heap 자료구조의 동작 원리를 애니메이션으로 시각화하는 교육용 웹 애플리케이션입니다.

## 목적

- 자료구조 학습자들이 Min Heap의 동작 원리를 직관적으로 이해
- 삽입(insert), 추출(extractMin) 연산의 단계별 과정 시각화
- heapifyUp, heapifyDown 알고리즘의 동작 방식 학습

## 주요 기능

### 1. 트리 뷰 시각화
- Min Heap을 이진 트리 형태로 표현
- 노드 간 비교/교환 과정을 하이라이트로 표시
- 부모-자식 관계를 선으로 연결

### 2. 배열 뷰
- 힙의 내부 배열 구조를 시각화
- 인덱스와 값의 매핑 관계 표시
- 연산 중인 인덱스 하이라이트

### 3. 단계별 애니메이션
- 각 연산을 단계별로 분해하여 표시
- 비교(compare) → 교환(swap) → 완료(complete) 과정
- 애니메이션 속도 조절 가능 (200ms ~ 1500ms)

### 4. 인터랙티브 컨트롤
- **삽입**: 원하는 값을 힙에 추가
- **최소값 추출**: 루트(최소값) 제거
- **초기화**: 힙 비우기
- **랜덤 채우기**: 5개의 랜덤 값 자동 삽입

### 5. 함수 설명
- insert, extractMin, heapifyUp, heapifyDown 함수 설명
- 각 함수의 시간 복잡도 안내

## Min Heap 특성

| 특성 | 설명 |
|------|------|
| 부모-자식 관계 | 부모 노드가 항상 자식보다 작거나 같음 |
| 구조 | 완전 이진 트리 (Complete Binary Tree) |
| 배열 구현 | 인덱스 i의 왼쪽 자식: 2i+1, 오른쪽 자식: 2i+2 |
| 시간 복잡도 | 삽입/추출: O(log n), 최소값 조회: O(1) |

## 대상 사용자

- 자료구조를 처음 배우는 학생
- 알고리즘 면접을 준비하는 취업 준비생
- 힙 자료구조를 복습하려는 개발자
