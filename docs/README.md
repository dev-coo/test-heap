## 빠른 시작

```bash
pnpm install
pnpm dev
```

http://localhost:5173 에서 확인

## 주요 기능

- 🌳 트리 뷰 시각화
- 📊 배열 뷰 시각화
- ▶️ 단계별 애니메이션
- 🎮 인터랙티브 컨트롤
- 📚 알고리즘 설명

## 기술 스택

- React 19 + TypeScript
- Vite 7
- Framer Motion
